Difference of a[{{-1}, {}}] and a[{-1, {}}]:¶

While these values have the same values the first is a 1x5 Tensor while the other is considered a 1-d Tensor of size 5. iTorch doesn't consider column vectors as N-D Tensors but as Tensors of size N for the 2nd command. Also index -1 is for the last line of the Tensor because the Tensor can be iterated from end to start by using negative indices.

Section 2 Basic implementations, visualizations and optimizations¶

Next, we will learn how to implement basic functions using Torch and Lua, find (local) minima using the optim package and visualizing the results. Let's assume we have the 1-d function

$$f(x)=x^2+\exp(|x|-x^2)$$

whose plot should look like (rendered with Wolfram Alpha):

$\DeclareMathOperator*{\argmin}{arg\; min}$ We want to find the location $x^*$, where $f(x)$ obtains its minimum value, i.e., $x^*=\argmin_{x} f(x)$. To do this, we will be using the optimization package optim and more specifically going to optimize the function using the L-BFGS algorithm. To find the minimum of the function $f(x)$ L-BFGS requires the gradient $\nabla_x f(x)$.

Section 3 Fitting a function to data points¶
In questions 2.1 you were asked to implement a function $f(x)$, its gradient $\nabla_x f(x)$, and compute the minimum of $f(x)$.
Now, let's assume that we are not given the function form, as in most cases in machine learning and applied statistics.
Instead, we are given a bunch of points in space and we want to fit a parameteric model, which explains it pretty well.
We are given the following point $(x, y)$ pairs.




We can see that the way the points are positioned on the plot, it looks a lot like our function $f(x)$ from question 2.1.
However, it's only the shape that looks similar, the points are located somewhere else in space and they are even upside down.
This means we need to fit our function $f(x)$ to our particular point set.
To this end wee parameterize the components that control the shape and position of the function,

$$
f(x; \theta)=\theta_0 + \theta_1 x^2+ \theta_2 \exp(\theta_3 |x| - \theta_4 x^2).
$$
Note that compared to the previous section, where the parameters where the inputs $x$.
In this example, however, $x$ are the observed variables.
Our parameters, instead, are the varables $\theta$.
Our goal now is to find the the optimal parameters $\theta^*$ for which we could a render $\hat{f}(x)$ function that would look plausible, given these points.


You can use the starter function "fit_simple_function.lua". The points for this dataset are stored in the file "points.dat".

Loss Function:¶
The most common loss function for seeing how our approximated function is fitting the data is the Mean Squared Error Function : $ \mathcal{L}(x;\theta) = \frac 1 2 \frac 1 N \sum_\tau^T(f(x;\theta) - t_\tau)^2 $ 

For convenience with the derivatives we use the $\frac 1 2$ in front of the function.

$$ f(x; \boldsymbol\theta)=\theta_0 + \theta_1 x^2+ \theta_2 \exp(\theta_3 |x| - \theta_4 x^2).$$
Partial derivatives for $\boldsymbol{\theta}$ of the function :¶
$${\nabla_{\theta_0} f(x; \boldsymbol \theta)}= {\partial f \over\partial \theta_0} = 1  \\ $$$${\nabla_{\theta_1} f(x; \boldsymbol\theta)}= {\partial f \over\partial \theta_1} = x^2  \\ $$$${\nabla_{\theta_2} f(x;\boldsymbol \theta)}= {\partial f \over\partial \theta_2} = \exp(\theta_3 |x| - \theta_4 x^2)  \\ $$$${\nabla_{\theta_3} f(x;\boldsymbol \theta)}= {\partial f \over\partial \theta_3} = |x| \theta_2\exp(\theta_3 |x| - \theta_4 x^2)\\$$$${\nabla_{\theta_4} f(x; \boldsymbol\theta)}= {\partial f \over\partial \theta_4} = -x^2\theta_2\cdot\exp(\theta_3 |x| - \theta_4 x^2)\\ 
$$
So the gradient of the Loss Function would be¶
$$\nabla_{\theta_0} \mathcal{L}(x;\boldsymbol\theta) = {\partial \mathcal{L} \over \partial \theta_0} = \frac 1 N \sum_\tau^T(f(x;\boldsymbol\theta) - t_\tau) \cdot {\partial f \over\partial \theta_0} = \frac 1 N \sum_\tau^T(f(x;\boldsymbol\theta) - t_\tau) $$$$\nabla_{\theta_1} \mathcal{L}(x;\boldsymbol\theta) = {\partial \mathcal{L} \over \partial \theta_1} = \frac 1 N \sum_\tau^T(f(x;\boldsymbol\theta) - t_\tau) \cdot x^2
$$$$\nabla_{\theta_2} \mathcal{L}(x;\boldsymbol\theta) = {\partial \mathcal{L} \over \partial \theta_2} = \frac 1 N \sum_\tau^T(f(x;\boldsymbol\theta) - t_\tau) \cdot \exp(\theta_3 |x| - \theta_4 x^2)   $$$$\nabla_{\theta_3} \mathcal{L}(x;\boldsymbol\theta) = {\partial \mathcal{L} \over \partial \theta_3} = \frac 1 N \sum_\tau^T(f(x;\boldsymbol\theta) - t_\tau) \cdot |x| \theta_2\exp(\theta_3 |x| - \theta_4 x^2) $$$$\nabla_{\theta_4} \mathcal{L}(x;\boldsymbol\theta) = {\partial \mathcal{L} \over \partial \theta_4} = \frac 1 N \sum_\tau^T(f(x;\boldsymbol\theta) - t_\tau) \cdot( -x^2\theta_2\cdot\exp(\theta_3 |x| - \theta_4 x^2))$$
